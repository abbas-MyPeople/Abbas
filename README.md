# Abbas Zoeb - Personal Portfolio

Personal website and portfolio of Abbas Zoeb - Founder & CEO of MyPeople, Philosopher & Visionary.

## Features

- Immersive dark-themed design with particle canvas background
- Interactive career roadmap/timeline with scroll-animated progress
- Project showcase with detailed descriptions and motivations
- Personal page exploring philosophy, interests, and identity
- Writing/blog section linking to Medium articles
- Fully responsive design
- Typewriter hero animation, scroll-reveal animations, counter animations

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Custom particle system with mouse interaction
- CSS Glass morphism effects
- Intersection Observer API for scroll animations
- Google Fonts (Inter, Space Grotesk, JetBrains Mono)

## Structure

```
/
├── index.html          # Main single-page site
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Particles, typewriter, animations
├── assets/             # Images and media (future)
└── README.md
```

## Run Locally

Simply open `index.html` in a browser, or serve with any static file server.
